Central processing unit

CPU

A Central processing unit, or commonly know as a CPU, carries out and controls the computer program's instructions by perfroming input/output operations, basic arithmetic and logic. An essential part of any computer, the CPU receives, directs and processes the computer's data.

Graphics card

GPU

A Graphics card, or commonly know as a GPU, is an expansion card for your PC that is responsible for rendering images to the display screen. GPUs are primarily used for gaming.

Motherboard

A motherboard is the main printed circuit board (PCB) in a computer. The motherboard is a computer's central communications backbone connectivity point, through which all components and external peripherals connect.

Random-access memory

RAM

Random access memory (RAM) is a computer's short-term memory, which it uses to handle all active tasks and apps. None of your programs, files, games, or streams would work without RAM.

Power supply

PSU

The power supply (PSU) is a critical part of any PC. It powers all the components in your PC, and a bad or faulty one can bring everything down.
